# Choparitte
subtitle: sleepless_kayoung

불쌍한 가영이를 위한 초파리 수면 분석 스크립트

## Prerequisite
- Python 3.x 
### Python packages
- `pandas`
- `PyQt5`
- `openpyxl`

## Usage
### For GUI
```commandline
python3 start.py
```
### For CLI
```commandline
python3 sleepless_kayoung.py <filename.xlsx> <sheet_name>
```

## Build
```commandline
pyinstaller choparitte.spec
```
